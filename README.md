# Personal portfolio site

Single-page site deployed via GitHub Pages at <https://leumasdam.github.io>.

## Edit

The whole site is in `index.html` — open in any editor, change the text,
push to `main`. GitHub Pages redeploys automatically (~1 minute).

## What to personalise

Search-and-replace these placeholders before showing the site to anyone:

| Placeholder | Where | Replace with |
|---|---|---|
| `Samuel` | Title, header logo, footer | Your first name |
| `Samuel · AI-assisted...` | `<title>` tag | Your full name + tagline |
| `linkedin.com/in/leumasdam` | Contact + nav | Your real LinkedIn URL |
| `damnwern@gmail.com` | Contact button | Your professional email |
| About section bio | Section `#about` | Your own one-paragraph story |
| Hero tagline | First `<h1>` | Refine to match the role you're targeting |

## Add a new project

Copy one of the `<article>` blocks under the `WORK` section and update the
text, links, and tech stack chips.

## Deploy

GitHub Pages serves whatever is on `main` of `leumasdam.github.io`. No build
step. Push to publish.

## Custom domain (optional)

When ready, add a `CNAME` file with your domain (e.g. `samuelnovak.dev`) and
configure DNS to point at GitHub Pages — see
<https://docs.github.com/en/pages/configuring-a-custom-domain-for-your-github-pages-site>.
