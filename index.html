Under the hood

Real Python, not glue code

Aggregation in the Excel Report Builder is written from scratch — grouping, totals and shares with the standard library, no pandas. Small, readable, and unit-tested.

app/aggregator.py 
def _group(records, key_fn, grand_total):
    """Group claim amounts by a key, with totals, averages and shares."""
    buckets = defaultdict(list)
    for record in records:
        buckets[key_fn(record)].append(record.amount)

    stats = []
    for key, amounts in buckets.items():
        total = round(sum(amounts), 2)
        stats.append(GroupStat(
            key=key,
            count=len(amounts),
            total=total,
            average=round(total / len(amounts), 2),
            share=round(total / grand_total, 4) if grand_total else 0.0,
        ))
    return sorted(stats, key=lambda s: s.total, reverse=True)
